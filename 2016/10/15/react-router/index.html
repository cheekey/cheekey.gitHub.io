react-router

react路由随手笔记

路由引用


import {Router, Route, hashHistory} from 'react'

route

路由定义通过route的path属性定义路由
Router 是容器，有个history属性可以定义路由的方式 


ReactDom.render((
  <Router history={hashHistory}>     
    <Route path='/' component={Index}/>       
    <Route path='/home' component={Data}/>     
  </Router>      
  ),document.getElementById('content')     
)

path属性定义路由
跟路由加载Index组件，/home加载Data组件，不单单加载Data组件，会先加载index组件然后再加载Data组件

IndexRoute

指定根路由加载的组件，根路由加载一个组件这个组件本身并没有东西，只负责让子组件加载进去


import React,{Component} from 'react'

export default class App extends Component{
  constructor(props){
    super(props);
    this.state={};
  }
  render(){
    return <div>{this.props.children}</div>
  }
}

我们用IndexRoute指定根路由组件，此时的this.props.children就是我们IndexRoute指定加载的组件。

IndexRedirect根路由路由重定向


ReactDom.render((
  <Router history = {hashHistory}>
    <Route path = '/' component={App}>
      <IndexRoute component = {Index}/>
      <IndexRedirect to="/po" />
      <Route path = '/po' component = {Data}/>
    </Route>
  </Router>
  ),document.getElementById('content')

这时根路由变成加载Data组件，to的值是路由；

Link

Link取代a标签，有to属性值是路由，跳转到那个路由
activeStyle属性指定路由样式
activeClassName指定路由的class 


render() {

    return (
      <div className="productBox">
        <div className='tab'>
          <div className='tab-left'><Link to='/ome' activeStyle={{color:'green'}}>路由ome</Link></div>
          <div className='tab-right'><Link to='/index' activeClassName={{main}}>路由index</Link></div>
        </div>
      </div>
    )
  }

IndexLink

链接到根路由/，不要使用Link，要使用IndexLink 


<IndexLink to='/'>到根路由</IndexLink>
